# NLP

A collection of educational NLP implementations with interactive visualizations.

## Projects

| Project | Description |
|---------|-------------|
| [**ngrams/**](ngrams/) | N-gram language models with smoothing and interactive tree visualization |
| [**tokenization/**](tokenization/) | BPE, WordPiece, and Unigram tokenizers with step-by-step visualization |

## Getting Started

Each project has its own README with setup instructions. Navigate to the project folder and follow the instructions there.

```bash
# Example: N-gram Language Model
cd ngrams
pip install -r requirements.txt
python run_server.py
```
